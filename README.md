# 🏠 Real Estate Consultant

A web application for managing real estate properties built with **Python** and **Django**.

## Technologies

- Python
- Django
- HTML5
- CSS3
- Bootstrap
- SQLite

---

## Features

- User Authentication
- Property Management
- Agent Management
- Add / Edit / Delete Properties
- Responsive Design
